# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.0] - 2025-01-XX

### Added
- Initial release
- Complete multiplayer turn-based strategy game
- Persian (Farsi) UI with RTL layout support
- Config-driven gameplay system
- WebSocket-based multiplayer (1v1)
- Path-based missile firing system
- Defense interception system
- Mana system
- Build and Battle phases
- Semi-realistic graphics with Phaser 3
- Sound system (with placeholders)
- Comprehensive logging system with toggle capability
- Version management system

### Technical
- Node.js server with Express and WebSocket
- React + Phaser client
- Modular architecture
- Server-side authoritative game logic
- Client-side rendering and animations
